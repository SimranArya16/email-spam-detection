# Email Spam Detection BYOP Project

## Overview
This project is a **Bring Your Own Project (BYOP)** submission for [Your Course Name].  
It detects whether an email is **Spam** or **Not Spam** using **Machine Learning** (Naive Bayes classifier) and provides the **spam probability**.  

The project includes a **Flask-based web application** where users can input an email and get instant results with a modern, attractive UI.

---

## Features
- Detects Spam vs Not Spam emails.
- Shows spam probability percentage.
- Modern web UI with responsive design and attractive colors.
- Trained on a real email dataset (`spam_ham_dataset.csv`).
- Easy to run locally or deploy.

---

## Dataset
- The dataset used: [`spam_ham_dataset.csv`](archive (1)/dataset/spam_ham_dataset.csv)  
- Contains labeled emails (`spam` and `ham`) for training the model.
- Features used: email text only.
